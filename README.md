# ai-grievance-redressal
AI-powered citizen grievance redressal system for smart governance

## Problem
Citizen complaints are delayed due to manual classification and poor tracking.

## Solution
A web-based platform that accepts complaints via text, voice, and image.
AI automatically classifies and prioritizes issues for faster resolution.

## Features
- Text, voice, and image complaint submission
- AI-based auto-classification
- Priority assignment
- Admin dashboard for tracking

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- AI/ML: NLP + Image processing
- Database: Firebase / SQLite

## Demo
- Voice-to-text complaint
- Image-based issue detection
